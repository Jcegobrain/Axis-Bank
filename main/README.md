# Hub de Produtos Bancários

Estrutura inicial do projeto (front-end + back-end) pronta para subir em um repositório.

Instruções rápidas:
- Copie o .env.example para .env e ajuste as variáveis.
- Rode `docker-compose up --build` para executar localmente com Docker.
- Frontend: http://localhost:3000
- Backend: http://localhost:4000
